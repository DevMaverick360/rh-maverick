Integração Google Forms → Maverick 360

Modo plug-and-play: envie form_responses com todas as perguntas/respostas (JSON), cv_base64 e job_code ou job_id. O servidor infere nome, e-mail e telefone (inclua uma resposta com e-mail válido). Opcionalmente envie name e email explícitos.

1. URL da API

Endpoint POST Substituído automaticamente pelo domínio atual ao abrir esta página pelo site. 
https://seu-dominio.com/api/cv/submit

2. Headers

Autenticação Defina CV_INGEST_SECRET no servidor e envie o mesmo valor. 
Content-Type: application/json
Authorization: Bearer <CV_INGEST_SECRET>

Alternativa:
x-cv-ingest-secret: <CV_INGEST_SECRET>

3. Variáveis de ambiente (servidor)

Referência .env
CV_INGEST_SECRET=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
OPENAI_API_KEY=
RESEND_API_KEY=

4. Corpo JSON

  • Plug-and-play: job_id ou job_code, e form_responses (ou form_submission) — ou name + email no JSON. CV (cv_base64 ou cv_url) é opcional.
  • Opcional explícito: name, email, phone (substituem ou complementam a inferência).
Exemplo JSON
{
  "job_code": "frontend-sp-2025",
  "form_responses": [
    { "question": "Qualquer pergunta", "answer": "Resposta" },
    { "question": "E-mail", "answer": "candidato@exemplo.com" }
  ],
  "cv_base64": "<opcional>",
  "cv_filename": "curriculo.pdf",
  "cv_mime_type": "application/pdf"
}

5. Google Apps Script

Gatilho instalável: função enviarMaverick (copie o nome do script guardado), origem Do formulário, evento Ao enviar o formulário. Erro "function not found": o gatilho não coincide com nenhuma função do projeto — apague o gatilho, guarde o código (Ctrl+S) e crie de novo. Não use onFormSubmit para HTTP.

Script modelo
/**
 * Gatilho INSTALÁVEL: função enviarMaverick — Do formulário — Ao enviar o formulário.
 */
const API_URL = 'SUBSTITUA/api/cv/submit';
const INGEST_SECRET = 'SUBSTITUA_PELO_SEGREDO';
const JOB_CODE = 'codigo-da-vaga';
const JOB_ID = '';

function enviarMaverick(e) {
  if (!e || !e.response) throw new Error('Gatilho: enviarMaverick, Do formulário, Ao enviar o formulário.');
  var itemResponses = e.response.getItemResponses();
  var formResponses = [];
  var cvBlob = null;
  var cvName = 'curriculo.pdf';
  var cvMime = 'application/pdf';
  for (var i = 0; i < itemResponses.length; i++) {
    var ir = itemResponses[i];
    var item = ir.getItem();
    var type = item.getType();
    var title = item.getTitle();
    if (type === FormApp.ItemType.FILE_UPLOAD) {
      var ids = ir.getResponse();
      if (ids != null && ids !== '') {
        if (Object.prototype.toString.call(ids) !== '[object Array]') ids = [String(ids)];
        if (ids.length) {
          var id0 = String(ids[0]);
          var m = id0.match(/[-\w]{25,}/);
          var fileId = m ? m[0] : id0;
          var file = DriveApp.getFileById(fileId);
          cvBlob = file.getBlob();
          cvName = file.getName() || cvName;
          cvMime = file.getMimeType() || cvMime;
        }
      }
      continue;
    }
    formResponses.push({ question: title, answer: formatAnswer_(ir.getResponse()) });
  }
  if (!cvBlob && !formResponses.length) throw new Error('Precisa de respostas (ex.: e-mail) ou ficheiro CV.');
  var payload = { form_responses: formResponses };
  if (cvBlob) {
    payload.cv_base64 = Utilities.base64Encode(cvBlob.getBytes());
    payload.cv_filename = cvName;
    payload.cv_mime_type = cvMime;
  }
  if (JOB_CODE) payload.job_code = JOB_CODE;
  if (JOB_ID) payload.job_id = JOB_ID;
  var res = UrlFetchApp.fetch(API_URL, {
    method: 'post',
    contentType: 'application/json',
    payload: JSON.stringify(payload),
    muteHttpExceptions: true,
    headers: { 'Authorization': 'Bearer ' + INGEST_SECRET }
  });
  if (res.getResponseCode() < 200 || res.getResponseCode() >= 300) {
    throw new Error(res.getContentText());
  }
}

function aposEnviarFormularioMaverick(e) {
  enviarMaverick(e);
}

function formatAnswer_(resp) {
  if (resp == null) return '';
  if (Object.prototype.toString.call(resp) === '[object Array]') return resp.join(', ');
  return String(resp);
}

6. curl

Linha de comando
curl -sS -X POST "https://SEU_DOMINIO/api/cv/submit" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer SEU_CV_INGEST_SECRET" \
  -d @corpo.json
Checklist
  • Migrações Supabase aplicadas (application_code, form_responses)

  • job_code da vaga igual ao enviado pelo script
